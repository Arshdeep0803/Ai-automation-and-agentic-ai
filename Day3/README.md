# Day 3 – Building Applications with Lovable and Replit

## What I Learned Today

Today, I gained hands-on experience in developing web applications using Lovable and Replit. This practical session helped me understand how AI-assisted development tools can be used to create useful applications quickly and efficiently.

## Projects Created Using Lovable

### 1. Daily Expense Tracker App

I developed a Daily Expense Tracker application that allows users to record and manage their daily expenses. The app helps track spending habits and provides a simple way to organize financial records.

### 2. Quiz App

I also created a Quiz Application using Lovable. The application presents questions to users, accepts their answers, and displays the results. This project improved my understanding of user interaction and application logic.

## Project Created Using Replit

### College Attendance Tracker

Using Replit, I developed a College Attendance Tracker application. The purpose of this application is to maintain attendance records of students and help monitor attendance percentages. Through this project, I learned how to structure data and build a practical educational tool.
