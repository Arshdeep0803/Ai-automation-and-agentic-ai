## Understanding AI Agents

An AI Agent is a system that can perceive its environment, make decisions, and perform actions to achieve a goal. AI agents use sensors to collect information and actuators to interact with their environment.

### Types of AI Agents

#### 1. Simple Reflex Agent

* Responds directly to current inputs.
* Uses predefined rules.
* Example: Automatic doors.

#### 2. Model-Based Reflex Agent

* Maintains an internal representation of the environment.
* Uses past information to make decisions.
* Example: Robot vacuum cleaners.

#### 3. Goal-Based Agent

* Works toward achieving specific goals.
* Evaluates possible actions before acting.
* Example: GPS navigation systems.

#### 4. Utility-Based Agent

* Selects actions that maximize overall benefit or utility.
* Considers different outcomes and chooses the best one.
* Example: Investment recommendation systems.

#### 5. Learning Agent

* Learns from experience and improves over time.
* Adapts to new situations.
* Example: Chatbots and recommendation systems.
